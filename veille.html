Veille Technologique

Surveillance des menaces, vulnérabilités et actualités cyber

Ma veille technologique est principalement axée sur l'écosystème Microsoft 365 — évolutions des services, nouvelles fonctionnalités, mises à jour de sécurité et vulnérabilités associées. En parallèle, je surveille les alertes CERT-FR / ANSSI pour rester informée des menaces actives, notamment celles susceptibles d'impacter les environnements Active Directory, Exchange et les solutions Microsoft déployées chez nos clients.

Mes outils de veille

📡

CERT-FR & ANSSI — Alertes & avis de sécurité

Les alertes et avis de sécurité du CERT-FR sont reçus quotidiennement via le groupe Teams de l'entreprise, permettant de centraliser les informations critiques en temps réel. Je suis également le compte @CERT_FR sur X (Twitter) pour recevoir les alertes instantanément.

📰

Feedly — Agrégateur de flux RSS

Feedly centralise plusieurs sources dans un seul fil : CERT-FR, Microsoft 365 Blog, Microsoft 365 Roadmap, Microsoft Dynamics 365 Blog et Microsoft 365 Security. Cela permet de suivre à la fois les vulnérabilités et les nouveautés produits Microsoft.

CERTFR-2026-AVI-0149 — Multiples vulnérabilités dans Microsoft Office

CVE-2026-21258 CVE-2026-21259 CVE-2026-21260 CVE-2026-21261 CVE-2026-21511 CVE-2026-21514 11 février 2026

Cet avis signale de multiples vulnérabilités découvertes dans Microsoft Office. Les bulletins de sécurité Microsoft référencent plusieurs CVE publiés le 10 février 2026. Dans le cadre de mon alternance chez Jesto Informatique, ce type d'alerte est directement actionnable : vérification des versions installées sur le parc client, application des correctifs via Microsoft Update ou déploiement GPO, et communication aux utilisateurs concernés.
